# Superpowers Adoption Roadmap

## Objective
Adopt a repeatable multi-agent workflow for this project with hard quality gates.

## P0 (This Week)
- Add repository workflow contract (`AGENTS.md`).
- Add spec/plan templates.
- Add local QA entrypoint scripts.
- Add minimum seat-map regression tests.

## P1 (1-2 Weeks)
- Add CI workflow for agent quality gate.
- Add PR template with verification evidence.
- Add skill-triggering smoke tests.

## P2 (2-4 Weeks)
- Add role-based agent chain (implementer/spec-reviewer/code-reviewer).
- Add pressure tests for documentation and planning behavior.
- Track metrics: rework rate, plan drift, first-pass PR success.

## Success Metrics
- Reduced regressions in seat-map pipeline.
- Higher share of changes with tests.
- Fewer hotfixes caused by skipped planning.
